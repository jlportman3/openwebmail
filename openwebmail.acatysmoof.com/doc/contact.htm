CONTACT

If you have encountered problems with OpenWebMail:

  • Search the Mailing Lists to see if someone else had the same problem and may have found a solution.

  • Check the Change Log to see if your problem has already been fixed.
    If it has been fixed, please install the Current Development Version until a release version is available.

  • Check the README and Frequently Asked Questions to make sure you followed the correct procedures.

If you want to seek help, please subscribe to the Mailing Lists and post your problem to the correct list.

If you want to submit patch or bug-report, please email it as a unified diff to the owm-devel mailing list, or to openwebmail.AT.acatysmoof.com.

A script is currently being developed for those who would like to mirror this site. Please stay tuned.

Please DO NOT email questions/problems to openwebmail.AT.acatysmoof.com or the authors directly. Please use the Mailing Lists so that the information could be shared by others.
