INTRODUCTION

OpenWebMail is a webmail system designed to manage very large mail folder files in a quick and memory efficient way. Robust addressbook, calendar, and webdisk systems are also included. OpenWebMail also provides a range of features to help users migrate smoothly from Microsoft Outlook.

OpenWebMail began based on Neomail version 1.14 from Ernie Miller in 2001.
