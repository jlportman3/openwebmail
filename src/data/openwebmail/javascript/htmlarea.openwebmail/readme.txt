
Files in this htmarea.openwebmail/ directory are customized version
of HTMLArea3.0 for Open WebMail.

If you are looking for the original HTMLArea3.0,
please check the files in ../HTMLArea-3.0-beta/ or visit their site at
http://www.interactivetools.com/products/htmlarea/

If you want to translate this customized editor for your language,

1. cd popups/YOUR_LANGUAGE
2. translate htmlarea-lang.js, insert_image.html, insert_sound.html
   and insert_table.html
