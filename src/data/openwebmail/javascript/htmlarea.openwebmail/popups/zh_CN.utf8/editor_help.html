编辑器说明

HTMLArea 是什么?

HTMLArea 是一个免费的 WYSIWYG 编辑器. 这个编辑器提供了以下功能:

  • 设定文字的 格式 为 粗体, 斜体, 或是 加底线.
  • 改变文字的 字体, 大小, 样式 或是 颜色.
  • 段落 靠左, 靠右 或是 置中对齐.
  • 建立无标号或是标号项目.
  • 段落 向内缩排 或是 减少缩排.
  • 插入水平分隔线.
  • 插入连结或是图片.
  • 检视编辑内容的 HTML 原始码.
  • 其他...

快速键

HTML编辑器提供的快速键如下:

  • ENTER -- 新段落
  • SHIFT-ENTER -- 换行
  • CTRL-A -- 全选
  • CTRL-B -- 粗体
  • CTRL-I -- 斜体
  • CTRL-U -- 底线
  • CTRL-S -- 删除线
  • CTRL-L -- 靠左对齐
  • CTRL-E -- 置中
  • CTRL-R -- 靠右对齐
  • CTRL-J -- 左右对齐
  • CTRL-1..6 -- 标题大小 (<h1> .. <h6>)
  • CTRL-Z -- 复原
  • CTRL-Y -- 取消复原
