編輯器說明

HTMLArea 是什麼?

HTMLArea 是一個免費的 WYSIWYG 編輯器. 這個編輯器提供了以下功能:

  • 設定文字的 格式 為 粗體, 斜體, 或是 加底線.
  • 改變文字的 字體, 大小, 樣式 或是 顏色.
  • 段落 靠左, 靠右 或是 置中對齊.
  • 建立無標號或是標號項目.
  • 段落 向內縮排 或是 減少縮排.
  • 插入水平分隔線.
  • 插入連結或是圖片.
  • 檢視編輯內容的 HTML 原始碼.
  • 其他...

快速鍵

HTML編輯器提供的快速鍵如下:

  • ENTER -- 新段落
  • SHIFT-ENTER -- 換行
  • CTRL-A -- 全選
  • CTRL-B -- 粗體
  • CTRL-I -- 斜體
  • CTRL-U -- 底線
  • CTRL-S -- 刪除線
  • CTRL-L -- 靠左對齊
  • CTRL-E -- 置中
  • CTRL-R -- 靠右對齊
  • CTRL-J -- 左右對齊
  • CTRL-1..6 -- 標題大小 (<h1> .. <h6>)
  • CTRL-Z -- 復原
  • CTRL-Y -- 取消復原

您可以在 HTMLArea 網頁 找到更多關於 HTMLArea 的資訊, 您也可在 HTMLArea 討論區中發表您的建議或是與其他使用者交換 HTMLArea 的使用心得.
