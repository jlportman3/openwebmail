Hjælp til editoren

Hvad er HTMLArea?

HTMLArea er en fri WYSIWYG-editor. Med den kan du formatere tekst og sende det som HTML:

  • Formatere teksten som fed, kursiv eller understreget.
  • Ændre skriftens skrifttype,størrelse, stil og farve.
  • Højrestille, venstrestille og centrere afsnit.
  • Lave lister med tal eller punkter.
  • Indrykke og tilbagerykke afsnit.
  • Indsætte en vandret linje.
  • Indsætte hyperlinks og billeder.
  • Vise den rå HTML-kode til det man er igang med at skrive.
  • og meget mere...

Tastaturgenveje

Editoren kender følgende tastaturgenveje:

  • ENTER -- nyt afsnit
  • SHIFT-ENTER -- linjeskift
  • CTRL-A -- marker alt
  • CTRL-B -- fed
  • CTRL-I -- kursiv
  • CTRL-U -- understreg
  • CTRL-S -- gennemstreg
  • CTRL-L -- venstrejuster
  • CTRL-E -- centrer
  • CTRL-R -- højrejuster
  • CTRL-J -- lige marginer
  • CTRL-1..6 -- overskrifter (<h1> .. <h6>)
  • CTRL-Z -- fortryd
  • CTRL-Y -- gentag

Hvor kan jeg få flere oplysninger og tale med andre HTMLArea-brugere?

Du kan finde flere oplysninger om HTMLArea på HTMLAreas hjemmeside, og du kan snakke med andre brugere af HTMLArea og sende kommentarer og forslag i HTMLAreas forum.
