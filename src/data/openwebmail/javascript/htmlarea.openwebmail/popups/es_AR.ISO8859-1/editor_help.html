Ayuda del Editor

¿Qué es HTMLArea?

HTMLArea es un editor libre WYSIWYG. Este potente editor le permite hacer lo siguiente:

  • Dar formato al texto con negrita, cursiva o subrayado.
  • Cambiar la tipografía, tamaño estilo y color.
  • Alinear a izquierda, centro, derecha o justificar párrafos.
  • Realizar listas numeradas o con viñetas.
  • Indentar párrafos.
  • Insertar líneas horizontales.
  • Insertar enlaces e imágenes.
  • Ver el código fuente HTML plano de lo que está editando.
  • y mucho más...

Atajos de teclado

El editor provee las siguientes combinaciones de teclas:

  • ENTER -- nuevo párrafo
  • SHIFT-ENTER -- salto de línea
  • CTRL-A -- seleccionar todo
  • CTRL-B -- negrita
  • CTRL-I -- cursiva
  • CTRL-U -- subrayado
  • CTRL-S -- tachado
  • CTRL-L -- alinear a la izquierda
  • CTRL-E -- centrar
  • CTRL-R -- alinear a la derecha
  • CTRL-J -- justificar
  • CTRL-1..6 -- encabezados (<h1> .. <h6>)
  • CTRL-Z -- deshacer
  • CTRL-Y -- rehacer
