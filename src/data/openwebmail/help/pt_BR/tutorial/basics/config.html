Open Webmail Ajuda e Tutorial >> Configurações Básicas

As configurações basicas permite que você altere várias preferências que seu e-mail necessita. Se você está se logando pela primeira vez no Open WebMail, você será redirecionado para a sua configuração de preferências, porém você poderá altera-las posteriormente. Este tutorial cobre as configurações básicas para pernonalizar seus webmail.

Essas são as principais opções de personalização:

  • Informações Pessoais
  • Prefêrencias de Visualização
  • Opções de Leitura
  • Operações com Mensagens
  • Opções Respondendo Mensagens
  • Opções de Filtros de Mensagens
  • Opções do Calendário
  • Opções Diversas

Para muitos usuários, o padrão de configurações é o suficiente. Se você não quiser alterar as configurações neste momento, então clique no botão salvar que está no final da página.

Seu administrador já deve ter pré-configurado essas opções para você.
