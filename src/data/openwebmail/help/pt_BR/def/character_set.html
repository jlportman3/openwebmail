Codificação de Caracteres

É uma lista definida dos caráteres reconhecidos pelo software e pelo computador. Cada carácter é representado por um número. O conjunto de carácter do modo ASCII, por exemplo, usa os números 0 a 127 para representar todos os caráteres ingleses e também caráteres especiais. O conjunto europeu de caráter de padrão ISO é similar ao ASCII, mas contêm caráteres adicionais para línguas européias.
