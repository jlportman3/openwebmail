Open Webmail 说明指南>> 使用通讯录

当然，一旦你已经设定通讯录时，你就会想要开始使用它。开始撰写一封新邮件 (查看 撰写邮件)。你就会看到那些可以根据你的需要而变更的标题栏位(收件者，主旨，等等)。

由于有如此多的电子邮件地址，当然你无法完全记得它们，这个时候通讯录就很有用了。经由点选 "通讯录" ( ) 按钮，你可以快速地增加一笔或是数笔地址到收件者栏位。一个新视窗会出现，带有所有的通讯录资料，每一笔资料都有一个选取方块。选取那些你想要寄出邮件的地址，并且点选继续。这些地址将会被放在收件者栏位中，经由点选相对应的通讯录图示，邮件地址就会被放在副本 CC 和密件副本 BCC 栏位中。点选 "Cancel" 将会关闭所有通讯录视窗且取消本次操作。

使用通讯录自动地将电子邮件地址加到外寄的邮件上，有第二种方法。假如你已经开启了通讯录编辑器，你可以点选任何一笔资料的电子邮件栏位，你将会被引导到新邮件画面，并且已经将该电子邮件放到收件者的栏位中。
