Open Webmail 说明指南>> 汇入与汇出通讯录

你也许已经储存了一份通讯录在你的电脑中，而不想在 Open Webmail 的通讯录中从头新增这些资料。这时候就是该使用 "汇入" 工具了，点选 汇入工具列按钮 ( ) ，系统将会开启通讯录汇入程式画面。汇入程式相容于 Outlook Express 5 (或更新的版本)，以及Netscape Mail 4.x ( "x" 表示次版本)。你需要开启你的离线式邮件阅读程式，汇出它的通讯录，存成 CSV (Comma Separated Values) 或是 TXT(ASCII Text File)格式。汇入程式画面也提供了简单的操步骤。 你必须从你的离线式邮件阅读程式中汇出通讯录，以便于汇入到 Open Webmail 的通讯录中。 (请参考你的离线式邮件阅读程式的使用手册以了解如何汇出通讯录)。当你正确地汇出你的通讯录并且存成适当的格式，你就可以开始使用 Open Webmail 来汇入通讯录了。要开始汇入通讯录，请点选 "浏览" 按钮，并且从先前储存在电脑中的目录中开启CSV 或是 TXT 格式的档案，当你找到该适当的档案时，接著点选 "开启"。最后，点选  "汇入" 按钮。此时 Open Webmail 上的通讯录内容就会和你的离线式邮件阅读程式中的通讯录内容完全一样了。

你也可以相反地操作，从 Open Webmail 中汇出通讯录，从离线式邮件阅读程式中汇入通讯录。点选"汇出通讯录" 按钮 ( )，它会引导出一个档案下载视窗，选择储存按钮以储存档案，依据你的需要输入档案名称，存档类型请选择 MicrosoftExcel Comma Separated Values File。你现在可以从你的离线式邮件阅读程式中汇入通讯录了，关于从文字档案汇入通讯录的操作步骤请参考你的离线式邮件阅读程式手册。
