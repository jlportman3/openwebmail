Open Webmail 说明指南 >> 编辑通讯录

通讯录是一个你常用邮件位址的小型的资料库，可以让你快速且容易地存取它。然而，通讯录有两种不同的使用模式。编辑模式，用来新增一笔纪录，删除，编辑，以及使用进阶功能。另外一种就是所谓的使用模式，可以在外寄邮件上立即地加入收信人的资讯(例如：收件者，CC 副本抄送，BCC 密件副本)。

当然，首先你需要先新增纪录。你必须先开启通讯录来编辑。只要在主要工具列点一下 ( ) 这个图示，就可以开启通讯录了。你会被导引到通讯录的编辑画面。

很明显地，你的第一步就是要新增一笔纪录。 (注意: 也许会出现些由系统管理者设定的一些纪录)在Name 文字栏位上轻点一下，并且输入必须的资料。使用 "Tab" 键可以直接跳到下一栏。你也可以输入昵称到 Note 栏位。当你输入完成后，点选 Add/Modify按钮，你现在就可以快速地使用这一笔邮件位址来寄信了。

当你完成所有邮件地址的建档后，也许会想要修正些资料。如想要编辑邮件地址资料，可以在名称栏位上点一下，那一笔资料就会出现在上方的编辑区域。就可以修改该笔资料，改成你要的样子。当你修改完成后，再点选一次 Add/Modify 按钮就可以了。

你可以直接点选右方的 Delete 按钮，删除你想要删除的项目。 你也许也想要删除所有的项目，但是分开一次一次删除会是枯燥无味的事。当然，你也可以点选  按钮，直接 "清除所有项目"。系统会要求你确认，点选 "OK" 就会删除所有项目、或是点选 "Cancel" 就会取消目前的操作。

你也可以把许多邮件地址加入到一个群组。使用一个邮件地址项目就可以让你寄信给许多已经存在的邮件地址，不需要一笔一笔地重新输入。 想要把许多邮件地址加入到群组中，点选群组 ( ) 就可以了。你会被导引到你的所有邮件地址的新视窗中，点选那些你希望加入到群组的项目，然后点选继续按钮。最后完成时再点选 Add/Modify 。
