搜寻工具列允许你搜寻目前所在资料夹的讯息。

使用这个工具列真的十分简单。 这个双向左箭头 ( )  将会完全跳到目前所在资料夹的第一笔讯息。请看 显示参考 来改变显示在每一页的讯息号码。

向左的箭头 ( ) 将会跳到讯息的前一页。

文字栏位 "1"，告诉你目前是在总页数当中的第几页. 你可以在目录夹中改变数字来跳到讯息的任何一页. 例如,如果写 "[Page 2 of 7]",你可以键入 "5"来代替 "2"，就会跳到第五页了。

向右箭,以及双向右箭完全和向左箭及双向左箭相反。 

你可以搜寻目前资料夹的讯息，藉由按钮"搜寻"旁边的文字栏位键入欲搜寻的字串，并按下搜寻. 你针对这封讯息是 从哪里来的、谁发出的、主旨、日期、附加的档案名称、信件表头、讯息的文字内容 或以上所有 等条件进行搜寻。
