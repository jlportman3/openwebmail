这是邮件工具列内的第二个小工具列：

多封邮件的状况下，在数字"1"旁会有向左和向右的方线指标。(向左指标代表要切换到前一封邮件，而向右指标代表要切换到下一封邮件)。  指标中的数字表示你目前看的邮件是第几封。 (1 表示是在该信件匣中的第一封邮件)。

在下拉式选单中的文字 选择回信底稿 ，可以让你选择回信时所要使用的格式。打开选单显示了你已经增加的格式 。当你选择一种格式之后，你将可以采用你选择的格式来回信。假如你还没有设定好任何格式， 点选这个图示 ( ) 后，你就可以开始编辑/或新增加一种格式。关于格式的定义，请查看 设定工具列。

* 假如你是在草稿匣阅读邮件 (查看 信件匣管理 )，这个按钮就不是要新设定一种格式，而是会编辑该封邮件。
