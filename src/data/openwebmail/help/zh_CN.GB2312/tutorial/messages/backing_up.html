Open Webmail 说明指南 >> 下载及备份

备份是说为了妥善保存，资料已经复制到另一个位置。你要周期性地对所有在你电脑上重要的档案和目录作备份，预防不可预期的问题发生。

你帐号上已经被指定了一些固定的空间来存放所有的档案，包括您的 Open Webmail 邮件。因此您也许最后会用完所有的空间(假如你已经很接近限定的容量，您将会收到一封邮件通知)。会要求您移除在Open Webmail 中的邮件。然而你也许拥有些重要的邮件想要保留，作为将来参考。你可以下载你的邮件到你的电脑上并且任何时后都可以使用文字编辑器来查看。

下载一个电子邮件资料夹:

你可以使用信匣管理(点选工具列中的 按钮)来下载你在 Open Webmail 中的电子邮件资料夹到你的电脑上。当你进入信件匣管理画面时，你可点选你想要下载的资料夹右方的磁片图示( )。就会开始下载到你的电脑上。系统会问你要如何处理下载的档案。点选"储存档案"选项，会出现一个弹出式档案浏览视窗，找到你要存放档案的位置，接著点选"储存"。

档会被储存为 .zip 格式， .zip 档案包含一个或数个被压缩的档案，你将会需要一个档案压缩/解压缩的工具程式来开启这些档案。下面列出一些可以用来压缩和解压缩  .zip 的档案 (注意这些和 Open Webmail并无相关):

  • Winzip
  • PicoZip
  • Free Zip Program
  • ZipWave

查看你的电子邮件:

当你已经下载了整个邮件资料匣，你可以使用上述任何一个程式来解压缩档案并且读取它。你无法经由连续点选两下来解压缩档案并且开启内附档案，相反的先开启文字编辑器 (如 Windows Notepad**)，并且使用档案功能表中的开启档案功能来开启该档案。所下载的邮件会在 e-mail headers 后以纯文字的格式表示。

不幸的是你所下载的邮件，会有好几行以无意义的文字出现导致你无法阅读。假如你用完了所有空间来储存你在Open Webmail中的邮件，但是又很坚持要求要继续存放使用，请和你的 系统管理者  联络，要求购买，增加你帐号上的空间。

*注意: 假如你使用 Microsoft Internet Explorer 5.5，你也许会再下载后要开启档案时会遭遇困难，他也许会存为 .pl 格式的档案。假如这样，使用上面的方法改存为 .zip 格式。

Windows Notepad 对于开启的档案大小有限制，假如档案太搭，你可以使用  Wordpad 来开启。当系统问你是否要继续开启时，回答 "Yes".。
