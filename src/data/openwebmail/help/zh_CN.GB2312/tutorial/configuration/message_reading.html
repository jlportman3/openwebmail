Open Webmail 说明指南 >> 读信相关设定

读信相关设定相当简单，改变收信的方法只要确定方块勾选或是不勾选就可以了。你可以依据你的使用偏好和最适萤幕宽度来设定。

读信相关设定对照表

项目/设定 说明/改变
读信时，使用固定宽度字型使用固定宽度的字型，某些人会觉得这样比较漂亮
读信时，使用笑脸图示使用笑脸图示(就像是用在e-mail或是信区的心情符号)显示在邮件中。
关闭邮件内的 JavaScript关闭邮件内支援 JavaScript 的功能，避免恶意的信件透过 JavaScript 干扰你的 Browser
关闭邮件内的 Embedded CGI关闭邮件内支援 CGI 的功能，避免某些广告信件透过此种方式确认使用者的读信动作
以超连结方式显示图片附件以超连结方式显示图片附件，节省读信所需的传输时间
